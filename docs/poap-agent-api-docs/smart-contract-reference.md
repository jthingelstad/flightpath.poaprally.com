# POAP Smart Contract Reference

The POAP contract address is the same across all networks:

```
0x22C1f6050E56d2876009903609a2cC3fEf83B415
```

To set the minting network via API, use `PATCH /drops/{dropId}/minting-config`. Ethereum Mainnet cannot be set through this endpoint.
